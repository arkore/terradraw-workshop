If you found any bugs or you have any idea to improve Terra Draw or the plugin, please create any issues in the repo.

## Contribution to Terra Draw community

Your support to Terra Draw is necessary to grow the community. If you like Terra Draw library, please star or fork repositories.

Please also feel free to make a pull request any time to contribute the projects.
